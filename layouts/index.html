Human-Centric Infrastructure üòç

Successful data science projects are delivered by data scientists who can experiment quickly and deliver production systems independently. Metaflow supports the full lifecycle of projects from prototype to production.

Doing this successfully requires a whole stack of infrastructure: data, compute, orchestration, versioning, etc. Metaflow helps you set up all of it, so data scientists can focus more on data science, less on engineering. 

                  
from metaflow import FlowSpec

class HelloFlow(FlowSpec):

    @step
    def start(self):
        print("hello world!")
        self.next(self.end)

    @step
    def end(self):
        print("done!")

HelloFlow()
